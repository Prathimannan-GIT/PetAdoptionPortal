Privacy Policy

How we collect, use, and protect information on the Pet Adoption Portal (frontend simulation).

Data usage

This project is a frontend-only demo. Data is stored locally in your browser via localStorage to simulate accounts, pets, and applications. In a real production deployment, all sensitive data would be processed securely on a server.

Information we collect

  • Account details: name, email, phone, city (as provided during registration)
  • Application details: housing type, experience, address, and a message to the shelter
  • Shelter details: shelter name, city, contact information
  • Pet profiles: pet details and image URLs submitted by shelters

How we use information

  • Adoption processing: share application details with the relevant shelter
  • Status updates: show pending/approved/rejected states
  • Safety: enable shelter verification workflows (admin approval)
  • Service improvements: improve clarity, UI and workflow (demo scope)

Data retention

In this demo, data persists in your browser until you clear site storage. If you want to reset the demo, clear browser storage for this folder origin.

Security measures

This demo does not implement server security. For real systems we recommend encryption in transit, password hashing, role-based server authorization, audit logs, and rate limiting.

Your rights

You can review the information you submit through forms and delete it by clearing localStorage data in your browser. For questions, use our Contact page.
